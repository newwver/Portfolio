[ featured - UI/UX개선 ]차세대 MIS 구축 프로젝트

1. 프로젝트 개요 목적 : 조직 내 정보 관리를 효율화하고, 더 정확하고 신속한 의사 결정을 과정을 지원하기 위함입니다.

2. 역할 및 책임 사용자가 업무를 효율적으로 수행할 수 있도록 사용자 경험을 고려하여 화면을 구성하고, 실제 화면으로 구현하는 역할을 맡았습니다.

3. 프로젝트 기간 및 팀 규모
  • 기간 : 프로젝트는 2023년 4월부터 2023년 7월까지 약 4개월간 진행했습니다.
  • 팀 규모 : 기간별 투입인원이 다르지만 대략 12명으로, PM, 개발자, 퍼블리셔 등으로 구성되었습니다.
4. 기술 및 도구
  • 기술 : 프로젝트는 Typescript 기반의 Angular 프레임워크를 사용하여 개발되었습니다.
  • 도구 : 형상 관리는 GitLab를 사용하였으며, 협업과 문서 공유는 Redmine를 활용했습니다.

5. 주요 성과 및 결과 주 사용자의 연령과 직군에 해당하는 가설을 설정하여 사용자 경험을 개선하였고, 개선된 화면을 통해 사용자는 작업상황을 한눈에 파악할 수 있게 되었습니다. 따라서 업무를 처리하는 속도가 빨라져 업무 효율이 향상되었습니다.

6. 향후 개선 방안 및 교훈 화면의 짜임새를 위한 고민부터 시작하면서 다양한 역할을 경험하였고, 이로 인해 더 폭넓은 역량을 발전시킬 수 있었습니다. 초기에 요구사항을 명확하게 정의하는 것과 사용자 피드백의 중요성을 다시 한번 확인했으며, 향후에도 초기에 요구사항을 명확히 수집할 것이고, 사용자와 자주 소통하며 진행할 것입니다.

7. 프로젝트 URL 보안상의 이유로 외부인 접근 불가.

[ featured - UI/UX개선 ]KPMS 시스템 고도화 프로젝트

1. 프로젝트 개요 목적 : 임대관리 · 시설관리 · 입주사서비스 시스템을 리뉴얼하여 사용자들의 편의성을 극대화 합니다.

2. 역할 및 책임 사용자가 업무를 효율적으로 수행할 수 있도록 사용자 경험을 고려하여 화면을 구성하고, 실제 화면으로 구현하는 역할을 맡았습니다.

3. 요구사항 분석 및 정의
  • 계약작성 및 관리 기능을 추가하여 계약자가 계약 진행상태를 직관적으로 확인할 수 있도록 합니다.
  • 전자세금계산서와 같은 형식으로 구현하여 누락되는 정보없이 명확하게 관리할 수 있도록 합니다.
4. 기술 및 도구
  • 기술 : 프로젝트는 Typescript 기반의 Angular 프레임워크를 사용하여 개발되었습니다.
  • 도구 : 형상 관리는 GitLab를 사용하였으며, 협업과 문서 공유는 Redmine를 활용했습니다.

5. 주요 성과 및 결과 시스템을 고도화함으로써 업무 처리 속도와 정확도가 향상되었고, 오류와 누락이 감소하였습니다. 모바일에서도 사용이 가능하도록 반응형 작업을 했기 때문에 언제 어디서든 시스템을 이용할 수 있게 되어 사용자의 만족도가 상승했습니다.

6. 향후 개선 방안 및 교훈 시스템의 특성과 개발 편의성을 고려하여, 더 나은 작업 방향을 결정하기 위한 고민이 필요했습니다. 어려운 상황에서도 자기주도적으로 문제를 해결해 나가는 능력이 중요한 프로젝트였습니다. 향후에도 사용자들의 의견을 수렴하여 사용자들이 시스템을 더 편리하게 이용할 수 있도록 할 계획입니다.

7. 프로젝트 URL 보안상의 이유로 외부인 접근 불가.
