차세대 MIS 구축 프로젝트

1. 프로젝트 개요 목적 : 조직 내 정보 관리를 효율화하고, 더 정확하고 신속한 의사 결정을 과정을 지원하기 위함입니다.

2. 역할 및 책임 사용자가 업무를 효율적으로 수행할 수 있도록 사용자 경험을 고려하여 화면을 구성하고, 실제 화면으로 구현하는 역할을 맡았습니다.

3. 프로젝트 기간 및 팀 규모
  • 기간 : 프로젝트는 2023년 4월부터 2023년 7월까지 약 4개월간 진행했습니다.
  • 팀 규모 : 기간별 투입인원이 다르지만 대략 12명으로, PM, 개발자, 퍼블리셔 등으로 구성되었습니다.
4. 기술 및 도구
  • 기술 : 프로젝트는 Typescript 기반의 Angular 프레임워크를 사용하여 개발되었습니다.
  • 도구 : 형상 관리는 GitLab를 사용하였으며, 협업과 문서 공유는 Redmine를 활용했습니다.

5. 주요 성과 및 결과 주 사용자의 연령과 직군에 해당하는 가설을 설정하여 사용자 경험을 개선하였고, 개선된 화면을 통해 사용자는 작업상황을 한눈에 파악할 수 있게 되었습니다. 따라서 업무를 처리하는 속도가 빨라져 업무 효율이 향상되었습니다.

6. 향후 개선 방안 및 교훈 화면 작업 과정에서 사용자 중심의 사고를 바탕으로 깊이 있는 고민이 필요했습니다. 프로젝트의 성공은 단순한 목표 달성뿐만 아니라, 실제 사용자의 경험에 의한 긍정적인 반응과 피드백에 있다는 생각을 했습니다. 앞으로도 사용자의 필요와 사용성을 충분히 고려하여 시스템에 반영할 것입니다.

KPMS 시스템 고도화 프로젝트

1. 프로젝트 개요 목적 : 임대관리 · 시설관리 · 입주사서비스 시스템을 리뉴얼하여 사용자들의 편의성을 극대화 합니다.

2. 역할 및 책임 사용자가 업무를 효율적으로 수행할 수 있도록 사용자 경험을 고려하여 화면을 구성하고, 실제 화면으로 구현하는 역할을 맡았습니다.

3. 요구사항 분석 및 정의(예시)
  • 계약작성 및 관리 기능을 추가하여 계약자가 계약 진행상태를 직관적으로 확인할 수 있도록 합니다.
  • 전자세금계산서와 같은 형식으로 구현하여 누락되는 정보없이 명확하게 관리할 수 있도록 합니다.
4. 기술 및 도구
  • 기술 : 프로젝트는 Typescript 기반의 Angular 프레임워크를 사용하여 개발되었습니다.

5. 주요 성과 및 결과 시스템을 고도화함으로써 업무 처리 속도와 정확도가 향상되었고, 오류와 누락이 감소하였습니다. 모바일에서도 사용이 가능하도록 반응형 작업을 했기 때문에 언제 어디서든 시스템을 이용할 수 있게 되어 사용자의 만족도가 상승했습니다.

6. 향후 개선 방안 및 교훈 시스템의 특성과 개발 편의성을 고려하여, 더 나은 작업 방향을 결정하기 위한 고민이 필요했습니다. 어려운 상황에서도 자기주도적으로 문제를 해결해 나가는 능력이 중요한 프로젝트였습니다. 향후에도 사용자들의 의견을 수렴하여 사용자들이 시스템을 더 편리하게 이용할 수 있도록 할 계획입니다.

APOLO-D Dashboard 구축 프로젝트.

1. 프로젝트 개요 목적 : 풀필먼트센터 내에서 발생하는 복잡한 물류 과정과 다양한 데이터·운영지표를 시각화하여 구현.

3. 프로젝트 기간

기간 : 프로젝트는 2021년 12월부터 2022년 6월까지 약 7개월간 진행했습니다.

2. 역할 및 책임 다양한 웹 브라우저와 모바일 기기에 대한 호환성 및 최적화 작업과 접근성 및 사용성(UX)을 고려하여 실제 화면으로 구현하는 역할을 맡았습니다.

4. 요구사항 분석 및 정의(예시)
  • 물류센터의 실시간 모니터링이 가능해야 하며, 병목현상 예상 시 즉각적인 경보 알림을 제공해야 함.
  • 센터의 현황은 자동 슬라이드로 표시되어, 추가 조작 없이도 모든 센터의 현황을 확인할 수 있어야 함.

5. 주요 성과 및 결과 디지털트윈 기술과 알고리즘을 활용하여 풀필먼트센터의 물류 기술을 고도화하였고, 실시간 모니터링을 통한 물류 흐름의 원활화와 빠른 병목 현상 해결이 가능해졌습니다. 시스템의 도입으로 소비자들은 당일 주문한 상품을 다음날 안정적으로 받을 수 있게 되었으며, 새벽배송 · 당일배송 등 시간대별 다양한 배송서비스가 하나의 풀필먼트센터에서 동시 구현이 가능해졌습니다.

6. 향후 개선 방안 및 교훈 물류기업의 경쟁력은 무형의 자산 기반으로 변화하고 있으며, 첨단 기술과 데이터의 활용이 기업의 역량을 높이는 데 있어 매우 중요해졌습니다. 축적된 물류 데이터를 활용하여 셀러들에게 마케팅 인사이트를 제공할 수 있듯이, 데이터를 더 효율적으로 활용하면 서비스 품질 향상에 도움이 될 수 있을 것입니다.
